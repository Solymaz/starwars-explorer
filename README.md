# Star Wars Explorer

Dive into the details of your favorite characters and movies from the iconic series.

## How to configure

npm install

## How to run locally

npm run dev

## How to build for production

npm run build

## How to run tests

npm test

## Technologies

- ReactJS
- NextJS
- Jest
- React Testing Library
- Material UI

## Requirements

Node v20.10.0 or newer
